## Features

- GitHub Flavored Markdown support
- Syntax highlighting
- Emoji Support
- Mermaid charts support

## How to Contribute

### Install dependencies

```
$ yarn install
```

### Run app

```
$ yarn start
```

### Package app

Builds app binaries.

```bash
$ yarn build
```
